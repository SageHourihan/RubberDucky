RubberDucky

RubberDucky is a responsive mobile first CSS framework that utilizes the 12 column grid. Based off of a simplegrid, RubberDucky's grid is easy to learn and offers simple, flat, minimalist styles to components. Unlike other frameworks, RubberDucky only has a few class that you need to learn to get your website up and running, making it a great framework for beginners or lightweight projects.

Breakpoints

RubberDucky has three breakpoints, breakpoint-small, breakpoint-med, and breakpoint-large. These breakpoints are used in the container class and the code is listed below.

.container {
  width: 90%;
  margin-left: auto;
  margin-right: auto;
  @media only screen and (min-width: $breakpoint-small) {
    width: 80%;
  }
  @media only screen and (min-width: $breakpoint-large) {
    width: 75%;
    max-width: 60rem;
  }
}

Grid

RubberDucky's 12 column layout is modeled after simplegrid with a few changes. RubberDucky simplified the class names. The grid uses a width of 96% and a gutter of 4% and everything in the container is set to a max-width of 960px but can easily be changed. The code:

<div class="container">
    <div class="row">
        <p class="col-1">col-1</p>
        <p class="col-2">col-2</p>
        <p class="col-3">col-3</p>
        <p class="col-4">col-4</p>
        <p class="col-5">col-5</p>
        <p class="col-6">col-6</p>
        <p class="col-7">col-7</p>
        <p class="col-8">col-8</p>
        <p class="col-9">col-9</p>
        <p class="col-10">col-10</p>
        <p class="col-11">col-11</p>
        <p class="col-12">col-12</p>
    </div>
</div>
